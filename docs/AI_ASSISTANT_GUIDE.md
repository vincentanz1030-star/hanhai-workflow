# AI 助手使用指南

## 功能概述

瀚海集团工作流程管理系统已深度集成 AI 助手（禾哲OpenClaw），提供智能预警、优化建议和实时咨询功能。

## 核心功能

### 1. 智能预警系统

AI 助手会自动监控项目状态，在以下情况发出预警：

- **任务超期预警**：任务已超过截止日期
- **即将到期预警**：任务将在3天内到期
- **项目延期预警**：项目整体进度低于预期
- **资源冲突预警**：同一岗位未完成任务过多（≥3个）
- **频繁催促预警**：任务被催促3次以上

### 2. 智能建议

AI 助手会根据项目状态提供优化建议：

- 优先处理超期任务
- 加强任务跟踪和沟通
- 评估岗位任务负荷
- 调整任务优先级

### 3. 全局 AI 助手

右下角悬浮的 AI 助手可以：

- 回答工作流程相关问题
- 分析项目状态
- 提供管理建议
- 上下文感知（知道用户在看哪个项目）

## 使用方法

### 查看预警和建议

1. 登录系统后，进入"数据看板"页面
2. 查看页面中间的"AI 智能洞察"卡片
3. 点击卡片可以展开/折叠详细信息
4. 点击预警右侧的 × 可以关闭单个预警
5. 点击建议按钮可以执行相应操作

### 与 AI 助手对话

1. 点击页面右下角的聊天图标
2. 在对话框中输入你的问题
3. AI 会基于当前项目上下文给出建议

**示例问题**：
- "帮我分析一下这个项目的风险"
- "这个项目的进度正常吗？"
- "有哪些任务需要优先处理？"
- "如何提高项目推进效率？"

## 配置 AI 助手

### 使用本地规则引擎（默认）

无需任何配置，系统会使用本地规则引擎提供基础的预警和建议功能。

### 集成 Coze AI（推荐）

1. 访问 [Coze 平台](https://www.coze.com/)
2. 创建或选择你的 Bot（禾哲OpenClaw助理）
3. 在 Bot 设置中获取 Bot ID 和 Token
4. 在环境变量中配置：

```bash
COZE_BOT_ID=your_bot_id_here
COZE_BOT_TOKEN=your_bot_token_here
```

5. 重启服务

### 配置步骤（生产环境）

1. 编辑 `.env.production` 文件
2. 添加 Coze 配置
3. 重新构建和部署

## 技术架构

### 前后端分离设计

AI 助手采用前后端分离架构，确保服务端环境变量的安全访问：

```
用户界面
    ↓
前端组件 (AIAssistant.tsx)
    ↓ fetch()
API 路由 (/api/ai/chat)
    ↓ callCozeBot()
服务层 (coze-service.ts)
    ↓ API Request
Coze AI 服务
    ↓ Response
用户界面显示
```

### API 接口

#### 1. 检查 AI 服务状态
```bash
GET /api/ai/chat
```

响应：
```json
{
  "configured": true,
  "message": "AI助手已配置"
}
```

#### 2. 发送聊天消息
```bash
POST /api/ai/chat
Content-Type: application/json

{
  "message": "请分析当前项目状态",
  "context": {
    "projectId": "xxx",
    "projectName": "示例项目"
  }
}
```

响应：
```json
{
  "success": true,
  "response": "AI 回复内容"
}
```

#### 3. 测试 Coze Bot 配置
```bash
GET /api/ai/test
```

响应：
```json
{
  "success": true,
  "config": {
    "configured": true,
    "botId": "7612859121276125222",
    "hasToken": true
  }
}
```

### 环境变量配置

AI 助手依赖以下环境变量：

```bash
# Coze API 配置
COZE_API_URL=https://api.coze.com
COZE_BOT_ID=7612859121276125222
COZE_BOT_TOKEN=your_token_here
```

这些配置在服务端安全存储，前端无法直接访问。

## 常见问题

### Q: AI 助手的预警准确吗？

A: AI 助手的预警基于实际项目数据和规则，具有很高的准确性。建议定期查看预警信息，及时处理风险。

### Q: AI 建议需要执行吗？

A: AI 建议仅供参考，是否执行取决于实际业务情况。建议结合实际情况和团队决策。

### Q: 如何关闭 AI 助手？

A: 目前不支持完全关闭 AI 助手，但可以忽略预警和建议。未来版本将提供开关选项。

### Q: AI 助手会消耗很多资源吗？

A: 不会。本地规则引擎几乎不消耗额外资源。配置 Coze AI 后，只在用户主动询问时才会调用 API。

## 最佳实践

1. **定期查看**：每天或每周查看 AI 智能洞察，了解项目风险
2. **及时响应**：对超期任务和严重预警要优先处理
3. **主动咨询**：遇到不确定的问题，主动向 AI 助手咨询
4. **反馈优化**：根据 AI 建议优化工作流程
