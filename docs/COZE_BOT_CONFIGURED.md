### 如果 AI 助手无法回复

**可能原因**:
1. Token 格式需要调整
2. API 地址需要确认
3. Bot 需要发布

**解决方法**:
1. 检查 Coze Bot 是否已发布
2. 确认 Token 是否正确
3. 联系 Coze 技术支持
